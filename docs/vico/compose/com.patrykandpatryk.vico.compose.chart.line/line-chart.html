Parameters

lines

the LineChart.LineSpecs to use for the lines. This list is iterated through as many times as there are lines.

minX

the minimum value shown on the x-axis. If not null, it overrides ChartEntryModel.minX.

maxX

the maximum value shown on the x-axis. If not null, it overrides ChartEntryModel.maxX.

minY

the minimum value shown on the y-axis. If not null, it overrides ChartEntryModel.minY.

maxY

the maximum value shown on the y-axis. If not null, it overrides ChartEntryModel.maxY.

decorations

the list of Decorations that will be added to the LineChart.

persistentMarkers

maps x-axis values to persistent Markers.

targetVerticalAxisPosition

if this is set, any com.patrykandpatryk.vico.core.axis.AxisRenderer with an AxisPosition equal to the provided value will use the ChartValues provided by this chart. This is meant to be used with com.patrykandpatryk.vico.core.chart.composed.ComposedChart.
