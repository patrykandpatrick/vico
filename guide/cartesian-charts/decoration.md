# Decoration

A decoration, represented by [`Decoration`](https://views.api.vico.patrykandpatrick.com/vico/views/com.patrykandpatrick.vico.views.cartesian.decoration/-decoration/), adds an additional layer of data to a chart. There are two built-in implementations—[`HorizontalLine`](https://views.api.vico.patrykandpatrick.com/vico/views/com.patrykandpatrick.vico.views.cartesian.decoration/-horizontal-line/) and [`HorizontalBox`](https://views.api.vico.patrykandpatrick.com/vico/views/com.patrykandpatrick.vico.views.cartesian.decoration/-horizontal-box/)—and you can add your own.

For an example, see the [“AI test scores”](https://github.com/patrykandpatrick/vico/blob/views/sample/charts/views/src/main/kotlin/com/patrykandpatrick/vico/sample/charts/views/AITestScores.kt) sample chart.
